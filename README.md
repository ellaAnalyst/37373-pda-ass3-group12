# 37373 PDA Assessment Task 3 – Group 12

## Project Title
Fields on the Rise: Do Fields of Study Align with Labour-Market Outcomes?

## Project Overview
This project investigates whether field-of-study participation aligns with labour-market outcomes across OECD countries. It compares education-side participation indicators with employment-side outcomes using two OECD datasets.

## Research Focus
- Which broad fields have gained or lost graduate share over time?
- Which fields show stronger employment outcomes?
- Which fields show the greatest mismatch between education participation and employment outcomes?

## Data Sources
This project uses two OECD datasets:
1. Distribution of enrolled students, new entrants and graduates by field of education.
2. Employment rates of tertiary-educated adults by field of study.

The datasets are not submitted to Canvas. They are referenced through source links and used for reproducible analysis.
